\section{Metodologia}
\label{chap:metodologia}

Para o Terceiro Trabalho Computacional, implementamos uma cadeia de processamento composta por três técnicas de reconhecimento de padrões, com o intuito de analisar como a redução de dimensionalidade e a flexibilidade das fronteiras de decisão influenciam a acurácia global e a estabilidade numérica. O protocolo experimental utiliza o dataset \textit{Wall-Following Robot Navigation} (24 sensores) submetido a 100 rodadas independentes de treinamento e teste.

\subsection{Redução de Dimensionalidade: PCA via SVD}
O PCA foi implementado através da decomposição em valores singulares (SVD) da matriz de covariância centralizada. O objetivo é projetar a matriz de dados $\mathbf{X} \in \mathbb{R}^{p \times N}$ em um subespaço $\mathbb{R}^{q}$ com $q < p$, mantendo a máxima variância explicada. Esta redução pode se tornar necessária para eliminar as redundâncias lineares, apesar da matriz de covariância estar bem condicionada, como observado no TC2, estabilizando as matrizes de covariância dos classificadores subsequentes.

\subsection{Classificador Quadrático Gaussiano (CQG)}
O CQG modela cada classe $w_i$ como uma densidade multivariada $\mathcal{N}(\boldsymbol{\mu}_i, \mathbf{C}_i)$. A função discriminante quadrática utiliza a distância de Mahalanobis e o log-determinante da matriz de covariância específica de cada classe. Embora a análise de condicionamento no TC2 tenha revelado que as matrizes originais deste dataset possuem posto completo e são bem-condicionadas, optamos por manter a regularização de Tikhonov ($\mathbf{C}_i + 10^{-6} \mathbf{I}$). Esta decisão visa garantir a robustez do modelo para projetos futuros e para cenários onde as amostras possam apresentar redundâncias colineares agudas, prevenindo a singularidade durante a inversão.

\subsection{Distância Minima ao Protótipo (DMP)}
O classificador DMP utiliza o algoritmo K-médias para identificar centros de massa locais. Para evitar o \textit{overfitting} e garantir que o modelo não degenere para um simples classificador 1-NN (onde cada amostra seria seu próprio protótipo), o intervalo de $K$ (número de protótipos) por classe foi delimitado como $K \in [2, 2\sqrt{N_{min}}]$, onde $N_{min}$ representa a quantidade de amostras da menor classe. O limite superior baseado na raiz quadrada é um critério heurístico para manter a parcimônia do modelo.

A seleção final de $K$ é feita por um processo de votação baseado em seis índices de validação de \textit{clusters}: Davies-Bouldin, Calinski-Harabasz, Dunn, Silhouette, I-Index e Ball-Hall. O algoritmo escolhe o $K$ que recebe a maioria dos votos dos índices. Em caso de empate na votação, o algoritmo seleciona o menor valor de $K$ entre os empatados, seguindo o princípio da navalha de Occam para priorizar o modelo mais simples que explique adequadamente os dados. Esta abordagem permite que o DMP capture a multimodalidade das classes, tornando um classificador não-linear potente mesmo utilizando a distância Euclidiana simples para a decisão final.
