# Documentação: Reconhecimento de Padrões - PPGETI

Esta documentação abrange os três Trabalhos Computacionais (TC) realizados para a disciplina de Reconhecimento de Padrões.

## Descrição Geral

O projeto contém a implementação de modelos de classificação, análise de dados e relatórios técnicos. Cada trabalho explora diferentes aspectos da área de reconhecimento de padrões, desde algoritmos básicos como KNN e DMC até técnicas mais avançadas.

## Conteúdo dos Trabalhos

- **Trabalho Computacional 1 (TC1):** Implementação de classificadores KNN, DMC e máxima correlação.
- **Trabalho Computacional 2 (TC2):** Focado em modelos de regressão e análise probabilística.
- **Trabalho Computacional 3 (TC3):** Clusterização e técnicas de aprendizado não supervisionado.

## Comandos Úteis

O `Makefile` na raiz do projeto contém comandos úteis para a gestão de dados e execução de tarefas comuns.
Utilize o `uv` para gerenciar as dependências e o ambiente virtual:

```bash
uv sync          # Sincroniza dependências
uv run jupyter lab # Inicia o Jupyter Lab para os notebooks
```
